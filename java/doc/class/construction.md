类的创建
========

1.	根据类名在常量池中定义一个类的符号引用。
2.	如果没有找到这个符号引用，说明类还没有被加载，则进行类的加载，链接，和初始化。
3.	虚拟机为对象分配内存（堆中）。
4.	将分配的内存初始化为零值（不包括对象头）。
5.	调用对象的<init>方法。

### 给对象分配内存

1.	指针碰撞

	堆内存中的空闲空间十分的规整，使用与未使用的空间全部为连续，只需移动一下指针就可以了。

2.	空闲列表。

	针对堆内存中的空间零散的存在，虚拟机维护着一个列表，记录着哪里被分配了，哪里还空闲
